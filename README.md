# 🚀 PLACIFY

**SUCCESS IS JUST NEXT STEP AWAY !!!**

PLACIFY is a premium, feature-rich SDE placement preparation companion and dashboard. Designed for aspiring software engineers, it streamlines daily routines, tracks DSA progress, schedules mock assessments, and provides a continuous SDE mentor in the form of **TechGuru**, a customized Gemini-powered AI assistant.

---

## ✨ Features

- **🤖 TechGuru SDE Advisor**: A global AI assistant powered by Google Gemini, equipped with an intelligent multi-model auto-fallback system (recovering through Flash, Pro, stable, and beta versions) to ensure 100% uptime.
- **📅 Dynamic Timetable Adjustments**: Interactive controls to customize daily target hours. Choose a "Leave Day" to redistribute study loads evenly across remaining days automatically.
- **💻 Coding Profiles Tracker**: Live integrations for monitoring progress on GitHub, LeetCode, HackerRank, and CodeChef.
- **📝 SDE TV & Playlists**: Curated video lectures grouped into specific tech domains (DBMS, OS, OOPs, System Design) for targeted learning.
- **⚡ SQL Interactive Editor**: Write, test, and validate SQL queries against multiple levels of complexity with real-time feedback.
- **📅 Placement Drive Calendar**: Comprehensive calendar to schedule, log, and prepare for upcoming mock tests and placement drives.
- **📋 Daily Task Organizer**: Categorized checklist (In Progress, Done) to manage daily milestones.

---

## 🛠️ Tech Stack

- **Frontend Core**: React.js (Vite), Lucide Icons for high-fidelity micro-illustrations.
- **Styling**: Modern, responsive Custom HSL CSS Palette with backdrop filters and glassmorphic designs.
- **AI Backend**: Google Generative AI API (Gemini-1.5-flash / Gemini-1.5-pro / Gemini-pro models).
- **Data Layer**: Responsive Client-Side state engine backed by `localStorage` persistence.

## 🚀 Getting Started

### 📋 Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### ⚙️ Installation & Local Setup

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Nishthayadav20/PLACIFY.git
   cd PLACIFY
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run the Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```

---

## 🛠️ AI Key Integration Details

No complex Google Cloud configurations are needed. To connect your assistant:
1. Fetch a free API Key from **[Google AI Studio](https://aistudio.google.com/)** (works with standard `AIzaSy...` and new enterprise `AQ...` prefixes).
2. Paste the key inside the TechGuru sidebar settings drawer in PLACIFY.
3. The key is stored locally in your browser's secure cache for future sessions.
